Información del proyecto

Proyecto final Lenguaje de marcas. Compuesto de 3 páginas: una de inicio con título, la modal de información, botón modo día/noche que guarda el cambio de estilos en localStorage, un input para mostrar un footer determinado y menú de navegación, una segunda página con el juego de tres en raya diseñado en javascript y una última página de tema libre donde se puede subrayar un texto eligiendo un color.
